# HealthTrack API

## Descripción del proyecto

HealthTrack API es una API REST desarrollada con Django y Django REST Framework.  
El proyecto permite gestionar pacientes y doctores dentro de un sistema básico de salud.

Cada paciente contiene información como nombre, edad y diagnóstico.  
Además, cada paciente está relacionado con un doctor asignado, quien tiene un nombre y una especialidad médica.

La gestión de datos se realiza únicamente mediante endpoints de API REST, sin utilizar Django Admin como interfaz de gestión.

---

## Tecnologías usadas

- Python
- Django
- Django REST Framework
- Django Filter
- SQLite
- Git y GitHub
